## 队列

> 只能在队尾进行插入，队首进行删除操作的结构

#### 队列的实现

```c
// 顺序队列
int* queue; // 存储数据的数组
int front, end; // 队头尾指针
int maxSize; // 最大容量

// 初始化队列
void initQueue() {
  queue = (int*)malloc(sizeof(int) * 100);
  if (queue == NULL) {
    printf("无法分配内存！！");
    exit(0);
  }
  front = end = 0;
  maxSize = 100;
}

// 判断队列是否已满
int isFull() {
  if ((end + 1) % maxSize == front) {
    return 1;
  } else {
    return 0;
  }
}

// 判断队列为空
int isEmpty() {
  if (end == front) {
    return 1;
  } else {
    return 0;
  }
}

// 入队
void insert_queue(int key) {
  if (isFull()) {
    printf("队满，无法插入!!");
    return;
  }
  end = (end + 1) % maxSize;
  queue[end] = key;
}

// 出队
void delete_queue() {
  if (isEmpty()) {
    printf("队列为空，无法完成出队操作!!");
    return;
  }
  front = (front + 1) % maxSize;
}
```
